## 如何使用 DNA 测试套件

1. 收集样本（如唾液或口腔拭子）。
2. 按说明注册你的采样盒以保护隐私，同时确保能够获取实验室结果。
3. 寄回样本，并等待约两至八周接收分析结果。

✅ 小贴士：为了保证准确性，请在采样前避免饮食或吸烟至少30分钟。
